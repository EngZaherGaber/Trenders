Introduction

Base URL: http://localhost:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

To authenticate requests, include an Authorization header with the value "Bearer {YOUR_AUTH_KEY}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by visiting your dashboard and clicking Generate API token.

Authentication

POST api/login

Example request:
curl --request POST \
    "http://localhost:8000/api/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"test@test.com\",
    \"password\": \"password\"
}"

const url = new URL(
    "http://localhost:8000/api/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "test@test.com",
    "password": "password"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

POST api/register

Example request:
curl --request POST \
    "http://localhost:8000/api/register" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "name=molestiae"\
    --form "email=sam.christiansen@example.org"\
    --form "password=password"\
    --form "type=company"\
    --form "category_ids[]=1"\
    --form "created_in=2023-09-28T12:16:54"\
    --form "address=illum"\
    --form "city=architecto"\
    --form "password_confirmation=password"\
    --form "image=@/tmp/phpSWnSJv" 
const url = new URL(
    "http://localhost:8000/api/register"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('name', 'molestiae');
body.append('email', 'sam.christiansen@example.org');
body.append('password', 'password');
body.append('type', 'company');
body.append('category_ids[]', '1');
body.append('created_in', '2023-09-28T12:16:54');
body.append('address', 'illum');
body.append('city', 'architecto');
body.append('password_confirmation', 'password');
body.append('image', document.querySelector('input[name="image"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());
